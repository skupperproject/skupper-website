Skupper community

Source code

Skupper is an open source project. The source is licensed under Apache License version 2.0 and hosted at GitHub under the skupperproject organization.

Mailing list

We use a mailing list hosted by Google Groups to answer questions and exchange ideas. Once subscribed, you can use the address skupper@googlegroups.com to send email to the list.

To subscribe, send an email to skupper+subscribe@googlegroups.com.

To unsubscribe, send an email to skupper+unsubscribe@googlegroups.com.

Issue tracking

We use GitHub issues to track bugs and enhancement requests.

If you run into a problem, start by searching for an existing issue:

If you can't find one that describes your problem or desired enhancement, create a new one:

Create an issue

Make sure to include details about your developer environment and Kubernetes variant. We especially appreciate issues with reproducer steps or programs.

Getting involved

The Skupper project welcomes new contributors. Here are some of the ways you can help:

  • Try Skupper and tell us about your experience
  • Fix bugs and make improvements
  • Write documentation
